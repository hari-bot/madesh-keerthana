# madesh-keerthana.com

Wedding invitation website for **Madesh & Keerthana** — 23 June 2026.

Built with React, Remix, TypeScript and Tailwind CSS.

## Features

- Personalised welcome modal via `?to=GuestName` URL param
- Hero section with responsive background
- Bride & Groom section
- Events section with reception & ceremony details
- Google Maps venue embed
- Countdown to the wedding day
- Pre-wedding photo gallery
- Background music player
- `/share` page to easily generate personalised invite links

## Development

Install dependencies:

```sh
npm install
```

Start the dev server:

```sh
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Generating Invite Links

Visit [http://localhost:3000/share](http://localhost:3000/share) to generate personalised invitation links for guests.
